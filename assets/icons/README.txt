ICONS FOLDER
=============

This folder is for any icon files you might need, such as:
- Dark mode toggle icon (moon/sun)
- Arrow icons for accordions
- External link icons
- "New" badge icons

You can use:
- SVG files (recommended - scalable, small file size)
- PNG files (if needed)

Alternatively, you might choose to use inline SVG in your HTML
or an icon library like Feather Icons or Heroicons.

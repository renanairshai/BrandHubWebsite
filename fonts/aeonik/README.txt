AEONIK FONT FILES
==================

Place your Aeonik font files in this directory.

You'll need these formats for best browser support:
- .woff2 (modern browsers, smallest file size)
- .woff (fallback for older browsers)

Typical font weights needed:
- Aeonik-Light.woff2 / .woff (300)
- Aeonik-Regular.woff2 / .woff (400)
- Aeonik-Medium.woff2 / .woff (500)
- Aeonik-Bold.woff2 / .woff (700)

Once you add the font files here, update the @font-face declarations
in /css/fonts.css to reference these files.
